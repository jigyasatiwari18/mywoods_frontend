# Mywoods - Timber & Design 🌲

Welcome to **Mywoods**, a modern web application designed for exploring premium timber collections, managing wood inventories via a built-in CMS, and connecting with timber design experts. 

## 🚀 Features

- **Dynamic Wood Catalog**: Explore rich wood tones, textured finishes, and premium selections.
- **Content Management System (CMS)**: Secure backend interface to easily add, edit, and delete wood inventory records.
- **Authentication**: JWT-based secure login system for CMS access.
- **Responsive Design**: Modern, glassmorphism-inspired UI tailored for desktop and mobile.

## 🛠️ Tech Stack

- **Frontend**: React (Vite)
- **Routing**: React Router DOM
- **Styling**: Vanilla CSS with Bootstrap integration
- **Icons**: Lucide React & React Icons

## ⚙️ Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/jigyasatiwari18/mywoods_frontend.git
   cd mywoods_frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Access the application:**
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📂 Project Structure

- `src/components/`: Reusable UI elements (Header, Footer, Cards).
- `src/features/`: Core feature components (Home, Woods Detail).
- `src/routes/`: Page-level components routing the application.
- `src/sampleData/`: Static data for initial rendering.
